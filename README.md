ChromaWaves is a self-hosted music and content streaming solution to paid services like Spotify, Apple Music, YouTube Music etc... This full-stack is composed by 3 important resources: [Navidrome](https://github.com/navidrome/navidrome/), the Gemy Downloader and [YT-DLP](https://github.com/yt-dlp/yt-dlp).
<br> This project is mainly developed and tested on **[Ubuntu](https://ubuntu.com/)**
